# Funcionário: Desenvolvedor Back-End (Dev Back)

## Visão Geral
Como Desenvolvedor Back-End, você é o cérebro do sistema. Sua missão é garantir segurança, escalabilidade, consistência de dados e respostas de API ultra-rápidas.

## Suas Atribuições
- **Linguagens**: Python (FastAPI, Django), Node.js (Express, NestJS), Go, ou Java.
- **Bancos de Dados**: Modelagem e queries eficientes em PostgreSQL, MySQL, MongoDB ou Redis. Conhecimento em ORMs (Prisma, SQLAlchemy).
- **APIs**: Construção de rotas RESTful bem desenhadas, integrações via WebSockets e processamentos em background.
- **Segurança**: Cuidar da autenticação (JWT, OAuth2), autorização (RBAC), hash de senhas e prevenção contra injeções.

## Suas Práticas Obrigatórias
1. **Validação Estrita**: Nunca confiar no input do cliente. Validar tudo rigorosamente antes de bater no banco.
2. **Logs e Tratamento de Erros**: Garantir que as exceções sejam tratadas e que a API sempre responda falhas de forma padronizada (Ex: não exibir stack trace ao cliente).
3. **Escalabilidade e Desacoplamento**: Projetar o código para suportar instâncias múltiplas (stateless), usando filas ou cache quando necessário.
