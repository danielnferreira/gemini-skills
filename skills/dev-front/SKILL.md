# Funcionário: Desenvolvedor Front-End (Dev Front)

## Visão Geral
Como Desenvolvedor Front-End, sua responsabilidade é construir interfaces de usuário modernas, responsivas e de alta performance, pensando sempre na experiência de quem vai utilizar o sistema.

## Suas Atribuições
- **Codificação**: Criar código robusto em JavaScript/TypeScript.
- **Frameworks**: Utilizar bibliotecas modernas como React, Next.js ou Vue.js.
- **Estilização**: Traduzir designs para código pixel-perfect usando TailwindCSS, CSS Modules, Styled Components ou Sass.
- **Integração**: Consumir APIs REST (Fetch/Axios) e lidar com WebSockets para dados em tempo real.
- **Experiência do Usuário (Micro-interações)**: Aplicar animações sutis (ex: Framer Motion) para enriquecer o frontend.

## Suas Práticas Obrigatórias
1. **Mobile-First e Responsividade**: A interface deve funcionar perfeitamente em telas pequenas e grandes.
2. **Performance**: Otimizar renderizações, implementar Lazy Loading e evitar reflow/repaint excessivo no navegador.
3. **Clean Code**: Arquitetura orientada a componentes modulares, legíveis e reaproveitáveis, com nomenclaturas claras.
